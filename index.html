Abstract

Emotional TTS in the user's voice

With the diagnosis of motor neuron disease and other diseases affecting speech, it is essential to ensure efficient and precise communication. In the UK, over 1500 people are diagnosed each year

Our solution to this involves the use of machine learning techniques to give the user the ability to add expressive emotions to the otherwise robotic voice in existing assistive systems.

This will work in conjunction with other technologies to enable effective communication for MND patients.

The Final Product

Key Features

Emotional TTS

Realistic voice modulation to reflect 7 emotions (joy, sadness, disgust, anger, fear, surprise, neutral) and natural speech patterns in Text-to-Speech to ensure efficient communications for MND patients.

Sentiment Analysis

Accurate analysis of the user's text to reflect the appropriate emotion in the TTS.

Easy to use and intuitive UI

Intuitive and user-friendly interface for easy navigation and control of the application, ensuring ease of use for MND patients once their motor functions deteriorate.

Project Introduction

Our project software consists of an offline web application. This application runs on a web browser in the user's computer via the localhost server. We made use of fully open source models throughout the project, and the end result is a user-friendly and natural text-to-speech (TTS) software with the ability to display emotion.
